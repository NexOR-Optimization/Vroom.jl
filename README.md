# MaxiCP.jl

[![Build Status](https://github.com/NexOR-Optimization/Vroom.jl/actions/workflows/ci.yml/badge.svg?branch=main)](https://github.com/NexOR-Optimization/Vroom.jl/actions?query=workflow%3ACI)

> [!WARNING]
> This packages is still a work in progress in early stage of development.

[Vroom.jl](https://github.com/NexOR-Optimization/Vroom.jl) is a wrapper for the
[VROOM solver](https://github.com/VROOM-Project/vroom).

## Installation

```julia
import Pkg
Pkg.add(url = "https://github.com/NexOR-Optimization/Vroom.jl")
```
